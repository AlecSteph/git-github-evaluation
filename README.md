Projet réalisé dans le cadre d'une évaluation Git & GitHub pour débutant.

# EXPLICATION ENTRE GIT ET GITHUB

Git: est un outil qui permet de suivre les modifications du code, de revenir en arrière et de travailler à plusieurs sur un même projet en local.

GitHub: est une plateforme en ligne qui héberge des projets Git et permet de partager le code, collaborer en équipe et sauvegarder les projets à distance.
